## 快速恢复环境

```shell

```

## Hexo 相关
```shell
# 创建文章： 
hexo new 文章名

# 清理数据
hexo  clean

# md生成新的静态页
hexo g

# 本地部署调试
hexo s

# Pages发布至指定仓库分支
hexo d
```
